# 基本設計書: 休暇申請サブシステム (V1.0)
作成日: 2018年4月1日

## 1. 概要
本システムは、従業員が休暇を申請し、管理者が承認するためのシステムである。

## 2. 画面定義 (Excel方眼紙風レイアウト)
| 項目名 | 必須 | 形式 | 備考 |
| :--- | :---: | :--- | :--- |
| 社員ID | 〇 | 半角英数 | ログインユーザーから自動取得 |
| 休暇区分 | 〇 | 選択 | 1: 有給休暇、2: 欠勤 |
| 開始日 | 〇 | YYYY/MM/DD | |
| 終了日 | 〇 | YYYY/MM/DD | |
| 申請理由 | | テキスト | |

## 3. 業務ルール (承認フロー)
- 全ての休暇申請は、**直属のマネージャー（MANAGER）**の承認が必要である。
- 申請日数が **5日を超える** 場合は、追加で**人事部（HR）**の承認が必要となる。

## 4. テーブル定義 (Leave_Request_Tbl)
| 物理名 | 論理名 | 型 | 桁数 |
| :--- | :--- | :--- | :--- |
| EMP_ID | 社員ID | VARCHAR | 10 |
| LEAVE_TYPE | 休暇区分 | CHAR | 1 |
| START_DT | 開始日 | DATE | |
| END_DT | 終了日 | DATE | |
| REASON | 理由 | VARCHAR | 255 |
| STATUS | ステータス | VARCHAR | 10 |
