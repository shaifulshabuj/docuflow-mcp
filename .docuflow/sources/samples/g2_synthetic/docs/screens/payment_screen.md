# 決済画面 詳細設計書

## 画面ID
SCR-PAY-001

## 概要
選択した商品の決済を行う画面。

## 画面項目
| 項目名 | 種別 | 必須 | 備考 |
|---|---|---|---|
| 決済金額表示 | ラベル | - | 読み取り専用 |
| クレジットカード番号 | テキストボックス | 〇 | ハイフンなし16桁 |
| 有効期限（月/年） | ドロップダウン | 〇 | |
| CVC | テキストボックス | 〇 | 3桁または4桁 |
| 決済実行ボタン | ボタン | - | |

## 状態遷移
- PENDING: 決済実行ボタン押下時
- SUCCESS: 決済完了時
- FAILED: 決済失敗時
※注意: 返金ステータスについては別画面(SCR-PAY-002)で扱うため、本画面の仕様には含まれない。
